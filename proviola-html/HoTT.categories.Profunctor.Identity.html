In particular the identity profunctor Id : C ⇸ C is represented by the identity functor and hence is given by the hom-functor C(−,−) : Cᵒᵖ × C → Set.

  Definition identity `{Funext} (C : PreCategory) : C -|-> C
    := hom_functor C.
End identity.
