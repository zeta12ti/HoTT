Section identity.
There is an identity functor. It does the obvious thing.
  Definition identity C : Functor C C
    := Build_Functor C C
                     (fun x ⇒ x)
                     (fun _ _ x ⇒ x)
                     (fun _ _ _ _ _ ⇒ idpath)
                     (fun _ ⇒ idpath).
End identity.
